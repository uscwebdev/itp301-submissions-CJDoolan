Project Summary: Food Festival App

Topic and Purpose:

The Food Festival Showcase App, developed using React.js on StackBlitz, serves as a mobile-centric platform for presenting and organizing a planned food festival taking place in LA. The site's main objective is to provide an intuitive experience for users interested in attending the festival. It caters to a broad audience of food lovers, aiming to attract individuals of all ages to participate in the event. The website is tailored for a diverse audience of food enthusiasts, encouraging their engagement and attendance at the food festival. The user-friendly design ensures accessibility for people of all ages, making it an inclusive platform for festival information and interaction.

How the Site Works:

The app's navigation is centered around a bottom navigation bar, guiding users to four main sections:

  • Home: Showcases promotional posters, creating an appealing visual introduction to the festival.
  • Menu: Displays various food categories and subsections that show the various vendors participating in the festival. This section also includes a search bar that allows users to find specific vendors or categories.
  • Map: Features a custom festival map for effective navigation within the event venue. Shop: Showcases festival merchandise available for purchase.

Additionally, a "More" button in the top left corner grants access to:

  • Socials: Links to the festival's social media profiles for updates and engagement. Partners: Information about event partners and sponsors.
  • Contact Us: A form for user inquiries or feedback, with all fields required for submission.

Extras Used:

The app incorporates several features that take into account users interaction with the website

  • React Library Integration: It utilizes the Router, Switch, Route, and Link elements from the react-router-dom library to navigate between different sections seamlessly.
  • UseState React Hook: Additionally, the site Implements the useState React Hook from react for local storage of user input. This is used for the search bar as well as more the user validation form, enhancing the interactivity and responsiveness of the app.
  • DOM Storage: Utilizes DOM storage to store information from the keyword search bar, enabling the preservation and retrieval of user inputs. When a keyword is searched this information will be updated to the DOM and it will lead to a specific page being called. If statements are used for the hardcoded keywords.
  • User Input Validation: Implements user input validation using the useState hook. If statements are used again to ensure that all required fields in the contact form are filled out before submission.

Conclusion:

The Food Festival Showcase App combines an engaging user interface with practical features, leveraging React.js and additional technologies to create a dynamic and user-friendly platform.
