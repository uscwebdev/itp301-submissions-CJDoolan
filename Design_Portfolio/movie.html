Movie Poster Redesign

This project served as a testament to my proficiency in both physical and digital art, as it demanded a blend of handcrafted artistic skills and digital design techniques. Moreover, it has ensured that I know the correct file formats and color profiles for print production, reinforcing essential skills for a seamless transition from design concepts to tangible, high-quality printed materials.

Problem Statement

This project aimed to create a promotional movie poster for a film that had already been released. In this project, students explored the interaction between image and text as well as experimented with large-scale printing
